Road Accident Dashboard

Analyze and explore road accident data to gain insights and inform decision-making for improved road safety.

Project Aim

The aim of this project is to analyze and explore road accident data to gain insights into the causes of accidents and identify strategies to mitigate future risks. By examining various factors contributing to accidents, the project seeks to understand key trends and inform decision-making for improved road safety.

Data Description

The dataset used in this project contains essential features related to road accidents, including:

  • Date of the incident
  • Severity of the accident
  • Light Conditions
  • Local Authorities
  • Number of Casualties
  • Number of Vehicles
  • Police Force Involved
  • Road Surface Conditions
  • Road Type
  • Speed Limits
  • Time of the incident
  • Urban Condition
  • Vehicle Type

Data Preparation

The data was prepared and modeled in Excel using Power Query. Key steps included:

  • Data Type Correction
  • Column Extraction
  • Renaming Columns
  • Data Cleaning
  • Data Loading
  • Power Query Automation

Data Analysis

During the analysis phase, we focused on exploring key columns to extract meaningful insights, leading to the identification of two types of Key Performance Indicators (KPIs):

Primary KPIs:

  • Total Number of Casualties
  • Number of Casualties by Severity: Categorized as Fatal, Serious, and Slight.

Secondary KPIs:

  • Total Casualties by Vehicle Type
  • Total Casualties by Road Type
  • Multiyear Trend Analysis
  • Total Casualties by Urban/Rural Area
  • Total Casualties by Light Condition
  • Total Casualties by Road Surface
